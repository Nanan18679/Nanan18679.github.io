Projects

During my time at UC Irvine, I have worked on various projects. Here are some of them.

LinkedIn Job Posting Salary Classification Analysis

This project analyzes job postings on LinkedIn (Kaggle dataset) to classify salaries into high or low tiers. It uses features like experience level, company size, remote status, and skills. Applied models include decision trees and random forests, enhanced with SMOTE for class balance.
Sun Country Airlines Business Analysis and Targeted Marketing Insights

Segmented Sun Country Airlines customers using K-Means clustering based on booking behavior, demographics, and travel patterns. Identified five customer groups to guide loyalty marketing and seasonal route planning. Tools used include Python, Scikit-learn, and Tableau.
Global Fertility Rate Analysis

Analyzed global fertility rates using World Bank data. Used linear regression to identify socio-economic factors like GDP and urbanization. Applied K-Means clustering to group countries with similar fertility patterns.
Heart Attack Prediction Analysis

Built models to predict heart attack risk using health and lifestyle data. SVM achieved the highest global accuracy, while k-NN performed well in regional models like Japan and Italy.
Y Combinator Database

Designed and queried a relational database containing Y Combinator startup data to uncover trends in founder backgrounds, funding status, and industry success. Provided insights for venture investment decision-making.
Twitch Influencer Network Analysis

Used Social Network Analysis on Twitch data to identify key influencers and calculate ROI using centrality scores and view metrics. Provided marketing insights through visualizations and profit modeling.
